Options

  • 'color' : String (Default: '#feed56')

    Sets the color of notes.

  • 'insertTargetText' : Boolean (Default: true)

    If true, the target's text will appear inside the note.

  • 'defaultText' : String (Default: 'Hello, NotesJS is awesome.')

    If 'insertTargetText' is set to false, the value of 'defaultText' will appear in the note.

  • 'confirmDelete' : Boolean (Default: false)

    If set to true, deletion of a note must be confirmed.
